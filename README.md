# Matching Lavoro–Candidato  
Progetto di Intelligenza Artificiale

Questo progetto implementa diversi approcci di AI nel dominio del matching tra candidato e offerta di lavoro:

-  Apprendimento supervisionato  
-  Apprendimento non supervisionato (clustering)  
-  Bayesian Network  
-  CSP (Random Walk & Simulated Annealing)  
-  Knowledge Base rule-based  

---

#  Requisiti

- Python **3.9 o superiore**
- pip aggiornato

- Crea ambiente Virtuale 

python -m venv venv
venv\Scripts\activate

- istallazione dipendenze

pip install -r requirements.txt 

- Esecuzione script

- Dalla cartella principale del progetto:

python src/nome_script.py
